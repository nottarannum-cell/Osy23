# Assets Folder

This folder contains all images and icons for the KnowYourHair website.

## Icons

All SVG icons are located in the `icons/` subfolder:
- `hair-strand.svg` - Decorative hair strand icon
- `shampoo.svg` - Shampoo bottle icon
- `oil-droplet.svg` - Oil droplet icon
- `brush.svg` - Hairbrush icon
- `float-test.svg` - Float test illustration
- `spray-test.svg` - Spray test illustration

## Hero Image

**IMPORTANT**: Replace this placeholder with the actual hero image from the PPT demo slides.

1. Extract the hero image from the PPT file
2. Save it as `hero-image.jpg` in this folder
3. Ensure it's optimized for web (recommended: max 1200px width, JPEG or WebP format)

If the image fails to load, the site will automatically show a gradient placeholder.
